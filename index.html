Acesso Remoto

Documentação: VM Windows para o Gateway do Power BI

Criar uma Máquina Virtual (VM) Windows capaz de executar o Gateway do Power BI.


Requisitos da Máquina:


  • Sistema Operacional: Windows (compatível com Windows 8 ou superior)
  • CPU: 8 Núcleos (Recomendado)
  • Memória RAM: Recomendado 8GB ou superior
  • Armazenamento: 50 GB
  • Conectividade: Acesso à Internet para download e registro do Gateway
  • Software: .NET Framework 4.8
